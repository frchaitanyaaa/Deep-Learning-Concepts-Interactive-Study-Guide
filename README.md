How to use it:

CNN and RNN/NLP tabs — switch between them
Click any question button (Q1–Q10) to open it
Each question has 4 tabs inside:

💡 Analogy — fun, real-world explanation to get the intuition first
📚 Concept — the actual theory, formulas, and key points
🗺️ Diagram — ASCII visual (exactly like your notes, but cleaner)
🎯 Quiz — test yourself! Getting it right auto-marks it as mastered


Track your progress bar at the top — go for 20/20! 🏆

Recommended flow: Analogy → Concept → Diagram → Quiz → Mark Done ✅
